TeX hyphenation patterns

This page is intended to summarize the situation of hyphenation patterns in current TeX distributions, in particular TeX Live. As of today, they are all bundled in a single package called hyph-utf8. If you're a pattern author and wish to update your patterns, please contact the hyph-utf8 package authours through the tex-hyphen mailing list.
